A bunch of automations that make my life easier.

```bash
chi 0.0.1

USAGE

  chi <command> [options]

COMMANDS

  backup folders              Compresses folders and copies them to a single destination. Executes `git fetch` first if they are git projects
  backup camera-uploads       Moves "Camera Uploads" photos and vidoes from Dropbox to multiple destinations. Organizes them into month-based subfolders
  help <command>              Display help for a specific command

GLOBAL OPTIONS

  -h, --help         Display help
  -V, --version      Display version
  --no-color         Disable colors
  --quiet            Quiet mode - only displays warn and error messages
  -v, --verbose      Verbose mode - will also output debug messages
```
